رنگ غالب شما: سفید

خانمی که ظاهری با رنگ غالب سفید دارد، ویژگی های برجسته ای هم داراست که اغلب عجیب و غریب به نظر می رسند! اگر موهایی نسبتا تیره، چشم های روشن و پوستی سفید (تا گندمی) دارید، رنگ های روشن برای شما بهترین خواهند بود.

تضاد شدید در رنگ های تیره و خیلی روشن پوست شما حتما شما را بی نظیر جلوه خواهد داد.

اعتماد به نفس داشته باشید و رنگ های روشن بپوشید تا رنگ بندی طبیعی شما خود را بیشتر نشان دهد.

میزان رنگ: رنگ های روشن

جواهرات: نقره بهتر است اما طلای براق هم خوب است.

بهترین رنگ های خنثی: سیاه، آبی سیر و خاکستری زغال سنگی

رنگ های مکمل: بنفش کم رنگ، قرمز، زرد، صورتی

سفید یا استخوانی؟! سفید

بهترین رنگ کیف و کفش: سیاه

بهترین رنگ برای ارایش: رنگ های شاد، بنفش و صورتی

مشاوره مو: هر چه طبیعی تر بهتر! روشن پوست ها هیچ وقت موهایتان را کوتاه نکنید. سعی کنید از براق کننده برای افزایش درخشندگی موهای خود استفاده کنید.

رنگ غالب شما: سرد

اگر رنگ های سرد، رنگ غالب در ظاهر شماست سعی کنید از رنگ هایی مثل بنفش کم رنگ، یاقوتی کبود و ترکیب رنگ قلع و سرب استفاده کنید تا بهترین ظاهر را داشته باشید.

لباس هایی با رنگ های خنثی بپوشید تا رنگ سرد شما بیشتر جلوه داشته باشد.

رنگ سرد شما نیازمند سایه ایست که ته رنگ آبی داشته باشد. هیچ وقت لباس هایی با رنگ قهوه ای یا مات نپوشید چرا که باعث می شود رنگ پریده به نظر برسید. آبی سیر بهترین رنگ برای شماست.

شدت رنگ: روشن تا نیمه روشن همراه با ته رنگ ابی یا صورتی

جواهرات: نقره یا طلای سرخ (ترکیب طلا و مس)

بهترین رنگ های خنثی: سیاه، آبی، خاکستری

رنگ های مکمل: آبی اقیانوسی، بنفش درباری، سرخابی، صورتی کم رنگ

سفید یا استخوانی؟! سفید

بهترین رنگ کیف و کفش: سیاه یا زغالی

بهترین رنگ برای آرایش: صورتی، قرمز، سیاه

مشاوره مو: موهای طبیعی شما بهترین گزینه است، هیچوقت موهایتان را رنگ نکنید.

رنگ غالب شما: گندمی

اگر رنگ ظاهر غالب شما گندمی است، نیازمند سایه هایی از رنگ های زیبا خواهید بود! میتوانید این رنگ بندی را با رنگ های روشن جهت ایجاد تضاد استفاده کنید.

از رنگ هایی با توناژ ضعیف دوری کنید! سایه های کم رنگ در طیف رنگی شما باعث می شود خسته و رنگ پریده به نظر برسید.

به جای یک سایه کم رنگ صورتی، از بنفش پررنگ و سبز استفاده کنید.

شدت رنگ: رنگ های تیره و نیمه روشن، بعضا روشن

جواهرات: طلا و نقره

بهترین رنگ های خنثی: سیاه یا قهوه ای تیره

رنگ های مکمل: بنفش، قرمز، خرمالویی، آبی اقیانوسی

سفید یا استخوانی؟! هر دو

بهترین رنگ کیف و کفش: اگر رنگ موی شما قهوه ای نیمه روشن تا تیره است، برای کیف و کفش هم قهوه ای تیره استفاده کنید; اگر رنگ موش شما قهوه ای تیره و یا سیاه است، رنگ سیاه را انتخاب کنید.

بهترین رنگ ها برای ارایش: قهوه ای، خثی، تمشکی

مشاوره مو: مو های تیره طبیعی شما بهترین گزینه است بنابراین هایلایت های خیلی روشن استفاده نکنید. سایه های متوسط، کاراملی و شکلاتی بهترین رنگ برای شما هستند. هایلایت خود را در اطراف صورت خیلی کمرنگ استفاده کنید.

رنگ غالب شما: روشن

خانمی که رنگ ظاهر غالب او روشن است شکوهی طبیعی در ظاهر خود دارد. به دلیل این ترکیب رنگ استخوانی، طیف زرد بهترین رنگ برای ارایش و لباس های این دسته افراد است.

رنگ های روشن برای این افراد چندان مناسب نیستند. رنگ هایی مثل بنفش، آبی، زرد و سبز بهترین گزینه ها خواهند بود.

شدت رنگ: کم رنگ تا رنگ های نیمه روشن

جواهرات: طلا و نقره

رنگ های خنثی: زغالی، آبی سیر، خردلی

رنگ های مکمل: زغالی، آبی، یاسی، یاقوتی کبود

سفید یا استخوانی؟! هر دو مورد

بهترین رنگ کیف و کفش: چون موهای بلوند و یا سفید دارید قوانین کلی برای شما اعمال نمی شود ولی در عوض می توانید از رنگ های خردلی، خاکستری مایل به قهوه ای و یا زغالی استفاده کنید.

بهترین رنگ ها برای ارایش: صورتی، سرخ، هلویی و یا رنگ های خنثی

مشاوره مو: اگر میخواهید رنگ طبیعی موهای خود را جلوه ببخشید حتما موهایتان را هایلایت کنید. از یک رنگ برای تمام مو ها استفاده نکنید تا مصنوعی به نظر نرسد.

رنگ غالب شما: ملایم

اگر شما خانمی با ظاهر زیبا، ظریف و با لطافت هستید رنگ بندی شما نیازمند سایه هایی از رنگ های زیباست که می توانید با رنگ های روشن برای ایجاد تضاد و ترکیبی از رنگ های نیمه روشن بپوشید.

سایه هایی از خردلی، خاکستری و قهوه ای نیمه روشن رنگ های مناسبی برای لباس ها، کت و یا دامن شما می باشند.

طیف رنگ قرمز از نارنجی تا قرمز روشن، انتخاب بسیار مناسبی برای شماست.

از رنگ های خیلی تیره و سنگین دوری کنید که باعث می شود مسن به نظر برسید.

شدت رنگ: رنگ های نیمه روشن

جواهرات: طلا و نقره

بهترین رنگ های خنثی: زغالی، قهوه ای، نیمه روشن

رنگ های مکمل: زرد، آبی سیر، مرجانی، سبز نعنایی

سفید یا استخوانی؟! استخوانی

بهترین رنگ کیف و کفش: قهوه ای نیمه روشن

بهترین رنگ آرایش: رنگ های خنثی و قهوه ای با رگه هایی از صورتی یا هلویی

مشاوره مو: هایلایت به شما می آید اما رنگ هایلایت خود را زیادی روشن و یا زیادی تیره انتخاب نکنید. رنگ های نیمه روشن همیشه برای شما بهترین اند! چندین سایه از هایلایت و رنگ های کم رنگ تر به شما ظاهری فوق العاده می بخشد. رنگ مسی هم به شرط اینکه به سمت رنگ های گرم مایل نباشد خوب است.

رنگ غالب شما: گرم

اگر رنگ های گرم رنگ غالب در ظاهر شماست تمام رنگ های پاییزی به شما می آید! کافیست در یک روز پاییزی به طبیعت نظری بیندازید و ایده های بی نظیری برای لباس های خود پیدا کنید.

در بهار و تابستان هم می توانید از همین رنگ ها استفاده کنید ولی خیلی کم رنگ تر.

همچنین انتخاب زرد و هلویی هم در این دو فصل مناسب اند.

شدت رنگ: رنگ هایی با زمینه زرد، سایه های پاییزی

جواهرات: طلا و یا ترکیب طلا و نقره

بهترین رنگ های خنثی: هر ترکیبی از قهوه ای

رنگ های مکمل: زنگاری، شکلاتی، مسی، فیروزه ای
